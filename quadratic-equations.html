Quadratic Equations

Introduction

Quadratic equations appear when modeling phenomena like projectile motion, area optimization, and physics problems. Unlike linear equations, they include an x² term, creating two solutions in most cases. In this lesson, we'll explore three main solving methods with clear steps and examples.

1. General Form & Discriminant

The standard form of a quadratic equation is:

ax² + bx + c = 0

The discriminant D = b² - 4ac tells us how many real solutions exist:

Discriminant (D)Solutions
D > 0Two distinct real roots
D = 0One real (repeated) root
D < 0No real roots (complex)

2. Solving by Factoring

Factoring is often the quickest method when the quadratic splits cleanly into two binomials. Follow these clear steps:

  1. Standard form: Write the equation as ax² + bx + c = 0.
  2. Identify coefficients: Determine a, b, and c.
  3. Find m and n: Find two numbers m and n so that m·n = a·c and m + n = b.
  4. Split the middle term: Rewrite bx as m x + n x, giving four terms.
  5. Factor by grouping: Pair terms and factor each group, then factor out the common binomial.
  6. Solve for x: Set each factor equal to zero and solve.

Example: Solve x² - 5x + 6 = 0

  1. Identify: a=1, b=-5, c=6.
  2. Find m, n: a·c = 6, b = -5. Numbers: m = -2, n = -3 (since -2 + -3 = -5 and -2 · -3 = 6).
  3. Split: 
    x² - 2x - 3x + 6 = 0
  4. Group: 
    x(x - 2) - 3(x - 2) = 0
  5. Factor: 
    (x - 2)(x - 3) = 0
  6. Solve: 
    x - 2 = 0  ⇒  x = 2
x - 3 = 0  ⇒  x = 3

3. Quadratic Formula

Use when factoring is difficult or impossible:

Formula: x = [-b ± √(b² - 4ac)] / (2a)

Example: Solve 2x² + 3x - 2 = 0

  1. Identify a=2, b=3, c=-2.
  2. Compute discriminant: D = 3² - 4·2·(-2) = 9 + 16 = 25.
  3. Plug into formula:
    x = [-3 ± √25] / 4 = [-3 ± 5] / 4
  4. Solve: x = (2/4) = 0.5 or x = (-8/4) = -2.

4. Completing the Square

Transforms the equation into a perfect square trinomial:

  1. Move c to right: ax² + bx = -c.
  2. Divide by a (if ≠1).
  3. Add (b/(2a))² to both sides.
  4. Write left as (x + b/(2a))², solve by taking square root.

Example: Solve x² + 6x = 7

x² + 6x + 9 = 7 + 9
(x + 3)² = 16
x + 3 = ±4  ⇒  x = 1 or x = -7

5. Special Cases & Tips

  • Perfect Square: x² + 4x + 4 = (x + 2)².
  • Difference of Squares: x² - 9 = (x - 3)(x + 3).
  • Always simplify radicals if possible.
  • Check your solutions by plugging back into original equation.

6. Practice Problems

  1. Factor and solve: x² - 7x + 10 = 0.
  2. Use formula: 3x² - 2x - 1 = 0.
  3. Complete the square: x² + 4x - 5 = 0.

Answers: (1) x=2,5; (2) x=1 or x=-1/3; (3) x=1 or x=-5.
