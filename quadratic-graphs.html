Graphing Quadratic Functions

1. The Parabola and Its Components

The graph of a quadratic function f(x)=ax²+bx+c is a parabola. Key components:

  • Vertex: The peak or trough of the parabola.
  • Axis of Symmetry: Vertical line through the vertex, x = h.
  • Intercepts: Y-intercept at (0,c); x-intercepts where f(x)=0.
  • Direction: Opens up if a>0, down if a<0.

3. Finding the Vertex and Axis

From standard form ax²+bx+c, the axis of symmetry is at:

x = -b / (2a)

To find the vertex (h,k):

  1. Compute h = -b/(2a).
  2. Plug h into f(x) to get k = f(h).

4. Example: Graph f(x)=x²-4x+3

  1. Find axis: x = 4/(2)=2.
  2. Vertex: h=2, k=f(2)=2²-8+3=-1 → (2,-1).
  3. Y-intercept: (0,3).
  4. Roots: Solve x²-4x+3=0 → (x-1)(x-3)=0 → x=1,3.
Figure: Graph of f(x)=x²-4x+3
